Stock Market Intelligent Analytics Platform

A stock market analytics platform that combines MySQL, Python, and Machine Learning to analyze stock trends, calculate technical indicators, and generate Buy/Sell/Hold signals.

Features

- Historical stock data collection
- Technical indicators (RSI, MACD, Bollinger Bands)
- Trend classification
- Volatility analysis
- Buy/Sell/Hold signal generation
- Data visualization

Technologies

- Python
- MySQL
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- yFinance
